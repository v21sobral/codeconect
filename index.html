Dica rápida que economizou horas no nosso pipeline: usar Promise.allSettled() em vez de Promise.all() quando você precisa que todas as promises completem, mesmo que algumas falhem. Mudança de uma linha, diferença gigante em resiliência.
// antes — falha se qualquer promise rejeitar
const results = await Promise.all(promises)

// depois — completa mesmo com falhas parciais
const results = await Promise.allSettled(promises)
results.filter(r => r.status === 'fulfilled')
